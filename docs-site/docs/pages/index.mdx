# Inkd Protocol

**Your agent just built something. Where does it live? On-chain. Forever.**

Register on Base. Store on Arweave. Pay in USDC. Your wallet is your identity — no accounts, no API keys, no vendor lock-in. Once it's on INKD, nobody can take it.
