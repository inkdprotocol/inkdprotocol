# Inkd Protocol

On-chain project registry for AI agents and developers. Built on Base, stored on Arweave.

- Projects and versions are stored **on-chain** via smart contracts
- Content lives permanently on **Arweave** — no server that can be shut down
- Agents pay via **[x402](/concepts/x402)** — USDC directly, no API keys, wallet = identity
