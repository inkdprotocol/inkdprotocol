# Uploading

Everything you store goes permanently to Arweave and is registered on-chain on Base.

---

## Text

```
/upload_text
```

1. Give it a name (e.g. `my-notes`)
2. Paste the text
3. See cost + choose visibility
4. Done

---

## File (PDF, ZIP, image, code…)

Send any file directly to the bot. Telegram handles up to 50 MB.

1. Send the file
2. Give it a name
3. See cost + choose visibility
4. Done

---

## GitHub repo

```
/upload_repo
```

1. Give it a name
2. Paste a GitHub URL: `owner/repo` or `https://github.com/owner/repo`
3. Bot downloads the ZIP, shows size + cost
4. Choose visibility
5. Done

---

## Push a new version

Open `/my_projects` → tap **Details** on any file → tap **Add version**.

1. Enter version tag (e.g. `v1.2.0`)
2. Enter changelog (optional)
3. Choose text or repo
4. Done

### Auto-detection

If you upload something with the same name as an existing file, the bot asks:
- **Add version** to the existing file, or
- **Create new file**

---

## Public vs Private

Before every upload:

```
🌍 Public   🔒 Only me
✖️ Cancel
```

- **🌍 Public** — anyone with the link can read it
- **🔒 Only me** — encrypted with AES-256-GCM before upload. Only your wallet can decrypt.

See [Private Uploads](/concepts/private-uploads) for the full security model.
