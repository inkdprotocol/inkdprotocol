# Uploading

Everything you store goes permanently to Arweave and is registered on-chain on Base. Paid in USDC from your bot wallet.

---

## How to upload

Tap **⬆️ Upload a File** from the home screen, or just send a file directly to the bot.

---

## File (image, PDF, video, ZIP…)

Send any file directly to the bot — it's automatically detected.

1. Send the file
2. Enter a name
3. Enter a description (optional — tap ⏭ Skip)
4. Choose visibility: **🌍 Public** or **🔒 Only me**
5. Confirm cost → stored

Supports images, PDFs, videos, ZIPs, and more. Max 50 MB.

---

## Text / Note

Tap **📝 Text / Note** from the upload menu.

1. Enter a name
2. Enter a description (optional)
3. Paste the text
4. Choose visibility
5. Confirm → stored

---

## GitHub Repo

Tap **🐙 GitHub Repo** from the upload menu.

1. Send a GitHub username or profile link:
   - `inkdprotocol`
   - `github.com/inkdprotocol`
   - `owner/repo` (direct)
   - `https://github.com/owner/repo` (direct)
2. If you sent a username → bot lists all public repos as buttons (with ⭐ stars)
3. Tap a repo → bot downloads the ZIP automatically
4. See size + cost
5. Choose visibility
6. Confirm → stored

The project name is taken from the repo name automatically. No extra steps.

---

## Public vs Private

Before every upload:

```
🌍 Public   🔒 Only me
```

- **🌍 Public** — anyone with the Arweave link can read it
- **🔒 Only me** — encrypted before upload. Only your wallet can decrypt.

See [Private Uploads](/concepts/private-uploads) for the full security model.

---

## Cost

Every upload shows estimated cost before you confirm:

```
📦 1.2 MB
💵 0.0032 USDC
```

Pricing = Arweave storage cost + 20% protocol fee (minimum $0.10 USDC).
