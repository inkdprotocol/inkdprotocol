# Telegram Bot

[@inkdbot](https://t.me/inkdbot) is the simplest way to use inkd. No SDK, no CLI, no terminal.

Send a file or text. Choose public or private. Done — stored forever on Arweave, registered on Base, paid in USDC.

---

## Commands

| Command | Description |
|---|---|
| `/start` | Set up wallet — goes straight to your wallet if already connected |
| `/wallet` | Balance, address, QR code, and quick actions |
| `/deposit` | Get your deposit address + QR code to add funds |
| `/withdraw` | Send USDC or ETH to any address |
| `/upload_text` | Store text permanently |
| `/upload_repo` | Archive a GitHub repo permanently |
| `/my_projects` | See all your stored files |
| `/history` | Recent uploads |
| `/search <name>` | Find public files |
| `/export_key` | Export your wallet private key |
| `/tutorial` | Interactive 4-step tour |
| `/links` | Website, Twitter, buy $INKD |
| `/cancel` | Cancel current action |
| `/help` | Show all commands |

---

## Upload flow

Every upload follows the same simple steps:

1. Run `/upload_text`, `/upload_repo`, or send a file directly
2. Give it a name
3. Bot shows: file size + exact USDC cost
4. Choose visibility:

```
🌍 Public   🔒 Only me
✖️ Cancel
```

5. Bot stores it — progress shown live:

```
⏳ Storing your file...

✅ Uploaded
▪ Creating record ⏳
▪ Confirming on Base ○
```

6. Done:

```
✅ Stored forever!

📄 my-document.pdf
🌍 Public
🔗 ar://QmAbc123…

#12 · my-document
```

---

## Public vs Private

- **🌍 Public** — anyone with the Arweave link can view the content
- **🔒 Only me** — content is encrypted before upload (AES-256-GCM). Only your wallet can decrypt it.

See [Private Uploads](/concepts/private-uploads) for the full security model.
