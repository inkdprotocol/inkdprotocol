# Telegram Bot

[@inkdbot](https://t.me/inkdbot) is the simplest way to use inkd. No SDK, no CLI, no terminal.

Send a file or text. Choose public or private. Done — stored forever on Arweave, registered on Base, paid in USDC.

---

## Home screen

Every time you open the bot you see:

```
⬆️ Upload a File
💼 Wallet     📁 My Files
❓ Help        🎓 Tutorial
```

---

## Upload flow

1. Tap **⬆️ Upload a File** — or just send a file directly
2. Choose what to upload: file, text, or GitHub repo
3. Enter a name + optional description (text/file only)
4. Choose visibility: **🌍 Public** or **🔒 Only me**
5. Confirm cost → USDC deducted automatically from your wallet
6. Done

---

## GitHub Repo

Send a username or profile link — the bot lists all public repos as buttons. Tap one to upload it directly.

```
inkdprotocol       → lists repos
github.com/inkdprotocol  → also works
owner/repo         → direct upload
```

---

## Public vs Private

- **🌍 Public** — anyone with the Arweave link can view it
- **🔒 Only me** — encrypted before upload. Only your wallet can decrypt.

See [Private Uploads](/concepts/private-uploads) for the full security model.

---

## Pricing

| Action | Cost |
|---|---|
| Store file / text | from $0.10 USDC |
| Store GitHub repo | Arweave cost × 1.20 (min $0.10) |

Exact cost is always shown before you confirm. No surprises.

---

## Wallet

The bot creates and manages a wallet for you. Your private key is stored encrypted — only you can export it via `/export_key`.

Add funds: send USDC and a small amount of ETH (for gas) to your wallet address on Base.
