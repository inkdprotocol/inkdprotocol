# Wallet & Withdraw

Your wallet is your identity in inkd. It holds your USDC balance and signs every upload.

---

## Create a wallet

Run `/start`. If you don't have a wallet yet, tap **✨ Create Wallet**.

The bot generates an Ethereum wallet and stores the encrypted key in your session.

```
✅ Wallet created!

Your address:
0xAbCd…EfGh

🔑 Save your key — shown once:
[private key]
```

**Save the key immediately.** Anyone who has it can access your funds. The bot will not show it again (except via `/export_key`).

---

## Add funds

```
/deposit
```

Or tap **📥 Add Funds** in `/wallet`.

Shows your address + QR code. Send USDC on Base to start uploading.

**Get USDC on Base:**
- [Coinbase](https://coinbase.com) — buy and send to Base
- [bridge.base.org](https://bridge.base.org) — bridge from Ethereum

Minimum for uploads: ~$0.20 USDC

Quick actions: **📥 Add Funds · 💸 Send · 📁 My Files · ⬆️ Upload**

---

## Send funds

```
/withdraw
```

Or tap **💸 Send** in `/wallet`.

1. Choose **💵 USDC** or **⟠ ETH**
2. Enter recipient address (0x…)
3. Enter amount
4. Confirm:

```
📤 Confirm send

💵 25.00 USDC
→ 0x1234…5678

This cannot be undone.
```

Tap **✅ Send now** — transaction is broadcast on Base.

---

## Export your key

```
/export_key
```

Shows your private key after a confirmation step. Use this to import into MetaMask or any EVM wallet.

---

## Security

- Keys are encrypted with AES-256-GCM using a server-side secret
- Keys are never logged or sent anywhere
- `/export_key` requires a confirmation tap
- If you lose Telegram access, use the exported key to recover your wallet
