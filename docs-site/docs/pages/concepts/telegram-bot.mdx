# Telegram Bot

[@inkdbot](https://t.me/inkdbot) — a wallet-gated Telegram interface for inkd. No SDK, no API key, no browser. Just a bot.

## What it does

- **Wallet management** — create a bot-managed wallet or connect an existing one
- **File uploads** — send any file, image, video, or text directly to Arweave
- **GitHub repos** — archive any public repo from a username or URL
- **Project management** — browse, version, and share your projects
- **Cost estimates** — `/estimate 500kb` before committing any funds

## Getting started

1. Open [@inkdbot](https://t.me/inkdbot) → `/start`
2. Tap **🆕 New Wallet** to create a bot-managed wallet (or connect an existing one)
3. Fund your wallet with USDC on Base (minimum $0.10)
4. Send any file or use `/upload_text` to store content permanently

## Commands

| Command | Description |
|---|---|
| `/start` | Home screen — wallet setup or main menu |
| `/wallet` | View address, ETH and USDC balance |
| `/upload_text` | Upload text or notes to Arweave |
| `/upload_repo` | Archive a GitHub repo |
| `/my_projects` | Browse your projects |
| `/history` | Recent uploads with share links |
| `/search <name>` | Search public projects |
| `/estimate <size>` | Calculate upload cost before spending |
| `/export_key` | Export your private key (bot wallets only) |
| `/tutorial` | 4-step onboarding walkthrough |
| `/links` | Website, socials, buy $INKD |
| `/cancel` | Cancel any pending action |
| `/help` | Full command reference |

## How uploads work

1. Bot downloads or receives your content
2. Uploads to Arweave via server wallet (instant, no gas needed from you)
3. Creates a project + pushes a version on Base via USDC payment
4. Returns the Arweave hash + Basescan transaction link

All uploads are registered on-chain under your wallet address. You own the record.

## Wallets

The bot supports two wallet modes:

**Bot-managed wallet** (recommended)
- Bot generates and encrypts a keypair locally
- You fund it with USDC on Base
- Bot signs all transactions on your behalf
- Export your private key anytime with `/export_key`

**External wallet** (read-only)
- Sign a challenge message from any Base-compatible wallet
- View your projects and history
- Cannot upload (bot needs to sign USDC payments)

## Privacy

Uploads can be **public** (visible to anyone via Arweave hash) or **private** (AES-256 encrypted before upload, only accessible with your wallet key).

## Inline mode

Search projects from any chat: `@inkdbot <project name>`
