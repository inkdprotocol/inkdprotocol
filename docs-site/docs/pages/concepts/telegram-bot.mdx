# Telegram Bot

Meet [@inkdbot](https://t.me/inkdbot): a wallet-gated chat interface that lets anyone push content to inkd without touching the SDK or API.

## Highlights
- **WalletConnect login** – tap the button, scan the QR (desktop) or deep-link (mobile), approve once. The bot remembers your wallet via SQLite sessions.
- **Paste-to-Ship** – `/upload_text` takes a project name and any text snippet (meme, note, code) and instantly stores it on Arweave + registers a project on Base.
- **Project overview** – `/my_projects` lists the latest inkd projects owned by your wallet (powered by the local indexer/SQLite).

## Step-by-step
1. Open [@inkdbot](https://t.me/inkdbot) → send `/start` → tap **🔗 WalletConnect**.
2. Telegram sends you a QR + a deep-link:
   - **Desktop users**: scan the QR from any WalletConnect wallet (Rabby, Coinbase Wallet, Rainbow, Ledger Live, etc.).
   - **Mobile users**: tap **Open Wallet** – your WalletConnect-compatible app opens directly.
3. Approve the session in your wallet. The bot automatically links the Base address and replies with `Wallet 0x… linked`.
4. Use `/upload_text` → enter a project name → paste content (&lt;50 MB). The bot returns:
   - `ar://…` hash + Arweave URL
   - Base transaction hash (createProjectV2)
5. `/my_projects` shows everything the same wallet has deployed (Telegram, SDK, or API – the indexer only filters by owner address).

> **Manual fallback:** Need to sign text manually (e.g., hardware wallet without WalletConnect)? Tap **✍️ Manual signature** to get the legacy challenge message, sign it once, and paste the signature back into Telegram.

## Deployment / Ops
The bot runs on `john-server` under **pm2** as `inkd-bot`.

```bash
ssh root@john-server
cd /root/.openclaw/workspace/agents/telegram-bot
pm2 status inkd-bot             # uptime & status
pm2 logs inkd-bot --lines 50    # inspect logs
pm2 restart inkd-bot            # manual restart
```

Configuration (`agents/telegram-bot/.env`):
- `TELEGRAM_BOT_TOKEN`
- `WALLETCONNECT_PROJECT_ID`
- `BOT_SERVER_WALLET_KEY` (this wallet is set as Settler on the Registry)
- `BASE_RPC_URL`, `INKD_REGISTRY_ADDRESS`
- `INKD_API_URL`, `INKD_INDEXER_DB`
- `SESSION_DB_PATH` (SQLite file for persistent logins)
