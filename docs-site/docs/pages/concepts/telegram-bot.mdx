# Telegram Bot

Meet [@inkdbot](https://t.me/inkdbot): a wallet-gated chat interface that lets anyone push content to inkd without touching the SDK or API.

## Highlights
- **Wallet-authenticated** – `/start` issues a one-time challenge. Once you sign it with your Base wallet, the bot knows which on-chain identity to use.
- **Paste-to-Ship** – `/upload_text` takes a project name and any text snippet (meme, note, code) and instantly stores it on Arweave + registers a project on Base.
- **Project overview** – `/my_projects` lists the latest inkd projects owned by your wallet (powered by the local indexer/SQLite).

## Step-by-step
1. Open [@inkdbot](https://t.me/inkdbot) → send `/start` → tap **Connect Wallet**.
2. The bot replies with a challenge like `inkd bot login for 7006655832 :: nonce abcd…`.
3. In your Base wallet (Coinbase Wallet, Rabby, Rainbow, etc.):
   - Open the menu → choose **Sign Message** (sometimes called *Verify Message* or *Sign Text*).
   - Paste the challenge text exactly as it appears.
   - Confirm the signature.
4. Copy the resulting signature (it starts with `0x…`) and send it back to the bot.
5. The bot replies with `Wallet 0x… linked`. From now on, `/my_projects` and `/upload_text` are unlocked for that wallet.
6. Use `/upload_text` → enter a project name → paste your content (&lt;50 MB). The bot returns:
   - `ar://…` hash + Arweave URL
   - Base transaction hash (createProjectV2)
7. `/my_projects` shows everything the same wallet has deployed (whether via Telegram, SDK, or API).

> **Tip:** A wallet only needs to link once. We now persist sessions in SQLite, so even after bot restarts the mapping stays intact.

## Deployment / Ops
The bot runs on `john-server` under **pm2** as `inkd-bot`.

```bash
ssh root@john-server
cd /root/.openclaw/workspace/agents/telegram-bot
pm2 status inkd-bot             # uptime & status
pm2 logs inkd-bot --lines 50    # inspect logs
pm2 restart inkd-bot            # manual restart
```

Configuration (`agents/telegram-bot/.env`):
- `TELEGRAM_BOT_TOKEN`
- `WALLETCONNECT_PROJECT_ID`
- `BOT_SERVER_WALLET_KEY` (this wallet is set as Settler on the Registry)
- `BASE_RPC_URL`, `INKD_REGISTRY_ADDRESS`
- `INKD_API_URL`, `INKD_INDEXER_DB`
- `SESSION_DB_PATH` (SQLite file for persistent logins)
