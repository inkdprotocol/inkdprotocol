# X Content Strategy — Inkd Protocol

---

## Posting Frequency

**Phase 1 (0-500 followers): Build credibility**
- 1-2 posts/day
- Focus on quality, not volume
- 1 thread/week max

**Phase 2 (500-1000 followers): Accelerate**
- 2-3 posts/day
- Threads 2x/week
- Active reply game (see below)

**Phase 3 (1000+ followers): Maintain**
- 3-5 posts/day
- Threads 2-3x/week
- Daily replies

6. @AIagents / AI agent founders on crypto Twitter — emerging niche, early mover advantage
7. Anyone asking "how do agents store data?" or "agent memory on-chain?" — intercept these

**How to reply:**
- Add information, don't just agree
- Bring a number or a concrete example
- Max 1-2 replies per thread from one account (don't spam)
- If someone is wrong about something in our domain, correct gently with data

**What not to do:**
- Don't reply to every post from big accounts (looks spammy)
- Don't self-promo in replies (mention inkd only if directly relevant)
- Don't start a reply with "actually" or "to be fair"

## Growth Tactics: 0 → 1000 Followers

**Week 1-2 (Foundation):**
- Post the pinned tweet first
- Follow 50-100 relevant accounts: agent builders, Base devs, infra founders
- Reply to 3-5 existing discussions per day before posting own content
- Don't post about inkd yet — post insights about the problem space

**Week 3-4 (Establish Voice):**
- Introduce inkd in context of the problem, not as a product announcement
- Post one thread explaining the problem (agent data dependency)
- Target: 5 replies from relevant accounts

**Month 2 (Momentum):**
- First build log thread
- Spotlight 1-2 builders using or interested in inkd
- Start tagging @jessepollak, @LitProtocol in contextually relevant posts
- Target: 100 followers

**Month 3 (Acceleration):**
- Post SDK code examples
- First "X inscriptions on testnet" milestone post
- Offer to join builders' projects as infra partner
- Target: 300-500 followers

**Pre-Mainnet (Launch ramp):**
- Teaser posts (no dates, just signals)
- SDK public release post with real code
- Thread: "why agent storage matters" — educational, not promotional
- Target: 1000 followers at mainnet

## Hook Formulas That Work in This Niche

**Formula 1: The Gap**
```
[existing solution] does X.
[existing solution] does Y.
nobody does Z.
that's the problem. [product] solves it.
```
*Why it works: creates tension, positions clearly without attacking competitors*

**Formula 2: The Specific Number**
```
[specific number] [thing] in [time period].
[implication in 1 sentence].
```
*Why it works: forces concrete thinking, builds credibility, algo loves engagement on factual content*

**Formula 3: The Honest Take**
```
real talk: [acknowledge the obvious criticism].
what's different: [concrete differentiator].
```
*Why it works: disarms skepticism, builds trust with technical audience*

**Formula 4: The Contrast**
```
[agent capability A].
[agent capability B].
[agent capability C].

but they cannot [basic thing that should be possible].

that's [product].
```
*Why it works: creates cognitive dissonance, memorable*

**Formula 5: The Build Log**
```
[what shipped] — [number].

[1-line what it means].

[link or next milestone].
```
*Why it works: authentic, shows traction, builders respect builders*

**Formula 6: The Future State**
```
imagine an agent that:
- [thing 1]
- [thing 2]
- [thing 3]

that's not a roadmap. that's inkd today.
```
*Why it works: sells the vision, grounds it in present reality*

## Content Rules Summary

- **Every post needs a first sentence that stops the scroll** — if it doesn't create tension, curiosity, or surprise in the first 10 words, rewrite it
- **Numbers over adjectives** — always
- **Lower case for takes and builds** — capitalized for official announcements
- **No hashtags** — ever
- **No emojis as openers** — kills credibility with technical audience
- **Threads should feel earned** — only write a thread when you have more to say than fits in one tweet, not for hype
