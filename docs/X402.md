# x402 Agent Payment Guide

inkd uses [x402](https://x402.org) — the open HTTP payment standard by Coinbase — for all write operations. No API keys. No accounts. Your wallet is your identity.

---

## How x402 Works

```
1. Agent sends POST /v1/projects
2. Server responds: HTTP 402 Payment Required
   Header: PAYMENT-REQUIRED → { price: "$0.001", network: "eip155:84532", ... }
3. Agent constructs payment signature using wallet
4. Agent resends request with: PAYMENT-SIGNATURE header
5. Server verifies via Coinbase facilitator
6. Request proceeds — payer address = project owner on-chain
```

One round-trip. Fully automated. No human needed.

## Quick Start

### Install

```bash
npm install @x402/fetch viem
```

### Agent example

```typescript
import { wrapFetchWithPayment } from '@x402/fetch'
import { privateKeyToAccount } from 'viem/accounts'
import { baseSepolia } from 'viem/chains'  // or: base for mainnet

const account = privateKeyToAccount(process.env.PRIVATE_KEY as `0x${string}`)

// Wrap fetch — auto-pays 402 responses using the agent's wallet
const fetch = wrapFetchWithPayment(account, baseSepolia)

// Register a project (pays automatically if 402 is returned)
const res = await fetch('https://api.inkdprotocol.com/v1/projects', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name:          'my-agent-tool',
    description:   'Does something useful',
    license:       'MIT',
    isPublic:      true,
    isAgent:       true,
    agentEndpoint: 'https://api.myagent.xyz/v1',
  }),
})

const { projectId, owner, txHash } = await res.json()
console.log(`Registered as project #${projectId}`)
console.log(`Owner on-chain: ${owner}`)  // = agent's wallet address
console.log(`TX: https://basescan.org/tx/${txHash}`)
```

### Push a version

```typescript
const res = await fetch(`https://api.inkdprotocol.com/v1/projects/${projectId}/versions`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    tag:         'v1.0.0',
    contentHash: 'ar://QmYourArweaveHashHere',
    metadataHash: '',
  }),
})

const { txHash, tag } = await res.json()
```

### Read (free — no payment)

```typescript
// Standard fetch — no payment needed for GET
const res = await fetch('https://api.inkdprotocol.com/v1/agents')
const { data } = await res.json()
// data = array of all registered AI agents
```

## Payment Details

| Endpoint | Price | Network |
|----------|-------|---------|
| `POST /v1/projects` | $0.001 | Base / Base Sepolia |
| `POST /v1/projects/:id/versions` | $0.001 | Base / Base Sepolia |
| All `GET` endpoints | Free | — |

Payments go to the inkd Treasury. Settlement is handled by the [Coinbase x402 facilitator](https://docs.cdp.coinbase.com/x402).

## What Happens On-Chain

When an agent pays via x402 and registers a project:

1. Server verifies payment via Coinbase facilitator
2. Server calls `InkdRegistry.createProject()` on Base
3. 1 $INKD is locked permanently in the Registry (removed from supply)
4. Payer's wallet address is recorded as the project owner
5. Project ID is returned — immutable on-chain proof of authorship

The agent's wallet address IS its identity. No username. No email. No OAuth.

## Local Development

Run the inkd API locally with x402 disabled (no payment needed):

```bash
cd api
cp .env.example .env
# Leave X402_ENABLED=false for local dev
npm run dev
```

Then make requests normally — no payment required in dev mode:

```bash
curl -X POST http://localhost:3000/v1/projects \
  -H "Content-Type: application/json" \
  -d '{"name":"test","description":"local test","license":"MIT","isPublic":true}'
```

Enable x402 in dev by setting:
```env
X402_ENABLED=true
SERVER_WALLET_ADDRESS=0x...   # receives payments
SERVER_WALLET_KEY=0x...       # signs on-chain txns
X402_FACILITATOR_URL=https://x402.org/facilitator
```

## OpenAPI / Types

Full OpenAPI spec: [api/openapi.yaml](../api/openapi.yaml)

### POST /v1/projects request body

```typescript
{
  name:          string   // 1-64 chars, unique across protocol
  description?:  string   // max 256 chars
  license?:      string   // default: "MIT"
  isPublic?:     boolean  // default: true
  readmeHash?:   string   // Arweave hash of README
  isAgent?:      boolean  // mark as AI agent
  agentEndpoint?: string  // agent's API endpoint URL
}
```

### POST /v1/projects/:id/versions request body

```typescript
{
  tag:           string   // version tag, e.g. "v1.0.0"
  contentHash:   string   // Arweave hash of the content
  metadataHash?: string   // optional Arweave hash of metadata
}
```

### Response (both endpoints)

```typescript
{
  txHash:      string   // on-chain transaction hash
  projectId:   string   // numeric project ID
  owner:       string   // wallet address of payer = owner
  signer:      string   // server wallet that signed the tx
  status:      string   // "success" | "reverted"
  blockNumber: string
}
```
