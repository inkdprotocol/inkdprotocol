### Architecture Diskussion (19:00–19:30)
**Identifizierte Lücken für GitHub-Replacement:**
1. Agent-Adresse geht on-chain verloren (pushedBy = Server Wallet statt Agent)
2. Project struct zu starr — fehlt: `accessManifestHash`, `forkOf`, `metadataUri`, `tagsHash`
3. Kein Metadata Standard definiert
4. Kein Indexer (The Graph) — bei scale bricht RPC zusammen
5. Multi-Wallet Encryption noch nicht gebaut (nur single-wallet AgentVault)

**Privacy Model:**
- Arweave ist immer öffentlich → Encryption before upload nötig
- Multi-Wallet: Symmetric Key + ECIES per authorized wallet = retroaktiv erweiterbar
- Access Manifest: on Arweave, hash on-chain in Project struct
